1. Conceitue as estruturas de SOs Monolítica, Em Camadas, Micronúcleo e Cliente-Servidor respectivamente:
As estruturas de SOs são diferentes modelos arquitetônicos usados na construção de sistemas operacionais. Monolítica é uma arquitetura em que todo o sistema opera em modo kernel, com grande complexidade e potencial de falhas. Em Camadas é dividida em camadas, onde cada uma delas é responsável por uma parte específica do sistema. Micronúcleo minimiza as funções do kernel, delegando a maior parte das operações para processos em espaço de usuário. Cliente-Servidor é baseado em comunicação entre processos onde clientes solicitam serviços que são atendidos por servidores.
2. Qual a linguagem empregada no sistema Unix? Em que ano?
O sistema Unix foi escrito em linguagem C, desenvolvido em 1972.
3. Quais foram os dois conceitos que o Unix lançou no mundo dos SOs? Explique-os.
Unix introduziu os conceitos de "arquivos como fluxos de bytes" e "sistemas de arquivos hierárquicos". O primeiro permite que todos os tipos de arquivos sejam manipulados de forma semelhante, enquanto o segundo organiza os arquivos em uma estrutura de diretórios em árvore.
4. Quais são os sistemas de arquivos usados pelo Unix e Windows?
Os sistemas de arquivos usados pelo Unix incluem ext3, ext4, enquanto o Windows utiliza NTFS e FAT32.
5. O que são GUIs, CLIs, TUIs, VUIs, NUIs, PUIs e BCIs?
GUI (Interface Gráfica do Usuário), CLI (Interface de Linha de Comando), TUI (Interface de Texto do Usuário), VUI (Interface de Voz do Usuário), NUI (Interface de Usuário Natural), PUI (Interface de Usuário Perceptiva), BCI (Interface Cérebro-Computador). Cada uma representa um método diferente de interação do usuário com um sistema operacional ou software.
6. O que são “sistemas do tipo Unix” e qual a diferença para "sistema Unix"?
"Sistemas do tipo Unix" referem-se a sistemas operacionais que seguem as especificações do Unix e compartilham suas características, mas não são necessariamente certificados como Unix. Um "sistema Unix" é certificado e cumpre rigorosamente com as normas e características do Unix original.
7. O que é um Firmware?
Firmware é um tipo de software que está gravado em um hardware e fornece controle de baixo nível para os dispositivos. Ele pode ser encontrado em tudo, desde PCs até eletrodomésticos.
8. O que é um sistema de operação em bando?
Um sistema de operação em bando processa grandes volumes de dados em lotes, permitindo que tarefas repetitivas sejam automatizadas e executadas sem a necessidade de intervenção humana.
9. O que é e como funcionam as Máquinas Virtuais?
Máquinas Virtuais (VMs) são ambientes de computação que emulam um computador físico. Funcionam executando um sistema operacional dentro de outro sistema operacional, permitindo a execução de múltiplos sistemas operacionais em um único hardware.
10. O que é RAID?
RAID (Redundant Array of Independent Disks) é uma tecnologia que combina vários discos rígidos em um único sistema para redundância de dados, aumento de desempenho ou ambos. Existem diferentes níveis de RAID, cada um com suas próprias características.
