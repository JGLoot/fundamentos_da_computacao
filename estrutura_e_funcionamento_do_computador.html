Estrutura e Funcionamento do Computador

Autores: Edson Lucas, João Gabriel, Matheus Kauê, Neemias Felipe e Welyngton Gonzaga

1. Quais são os cinto elementos essenciais para o funcionamento do computador? \/

R= Unidade aritmética-lógica (UAL / ULA), a unidade de memória, a unidade de controle, a unidade de entrada e a unidade de saída.

2. Com suas palavras, explique o que é e como funciona uma Unidade Aritmética-Lógica (ULA)? \/

R= É um programa que executa as principais operações lógicas e aritmética. Ela funciona como soma, subtração, divide e determina se um número é positivo ou negativo ou se é zero.

3. Com suas palavras, explique o que é e como funciona uma Unidade De Memória? \/

R= A memória armazena grupos de dígitos binários (palavras) que podem representar instruções a serem executadas pelo processador ou dados para serem operados pelas instruções durante a execução de um programa. Ela também serve como local de armazenamento para resultados intermediários ou finais obtidos pelas operações aritméticas efetuadas pela unidade aritmética-lógica. A memória é uma unidade que executa somente duas operações: leitura e escrita.

4. Com suas palavras, explique o que é e como funciona uma Unidade De Entrada? \/

R= É um dispositivo eletrônico que permite a entrada de dados no computador ou em um sistema de processamento de informação e funciona da seguinte forma. Essas unidades são muitos úteis para interação entre usuários e a máquina permitindo que comandos sejam enviados e informações sejam inseridas no sistema. Alguns dos dispositivos de entrada mais comuns são: teclados, discos magnéticos, fitas e conversores analógico-digitais (ADC´s).

5. Com suas palavras, explique o que é e como funciona uma Unidade De Saída? \/

R= É um dispositivo responsável de enviar dados processados pelo sistema para o monitor e funciona através do transporte de dados para outros dispositivos como impressoras alto falantes entre outros.

6. Com suas palavras, explique o que é e como funciona uma Interface? \/

R= Interface é o nome dado como ocorre na comunicação entre 2 partes distintas que não pode se conectar diretamente, ela pode funcionar da seguinte forma. Seus membros podem ser implementados por qualquer classe ou struct que implemente a interface. Um bom exemplo sobre interfaceamento é o caso do terminal de vídeo, que pode operar como dispositivo de entrada e de saída. Este terminal transmite e recebe dados serialmente, um bit de cada vez, enquanto a maioria dos computadores manipula dados em paralelo. Assim, o terminal de vídeo precisa de um circuito de interface que faça as modificações necessárias dos dados transmitidos/recebidos, de maneira a permitir que a comunicação terminal-computador e computador-terminal seja feita sem problemas.

7. Com suas palavras, explique o que é e como funciona uma Unidade De Controle? \/

R= Unidade de Controle (UC) assume toda a tarefa de controle das ações a serem realizadas pelo computador, ela funciona através da seguinte forma: gerencia o processamento de instruções e ordena o fluxo de dados dentro da CPU e entre outros componentes de computador.

8. Com suas palavras, explique o que é e como funciona um Processador (Unidade Central de Processamento)? \/

R= Serve para receber entrada de informações em forma de instrução de programas e executar trilhões de cálculos para fornecer a saída que o usuário interage com o dispositivo.

9. Com suas palavras, explique o que é e como funciona um Circuito Lógico? \/

R= Circuitos lógicos são combinações de componentes eletrônicos, sua função é realizar operações lógicas em um ou mais sinais de entrada para produzir uma saída.

10. Com suas palavras, explique o que é e como funciona uma Placa-Mãe? \/

R= A placa mãe é o componente responsável por abrigar e conectar todas as outras peças do seu computador e sua função é unir todas as partes do sistema numa só rede de fios porque dispõe de caminhos que permite a troca de informações entre processadores, memórias, placa etc..
