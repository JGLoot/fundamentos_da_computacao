1. Como converter de base 2 para base 8? \/

R= De 3 em 3 bits, pegue o número binário (base 2) e divida por 8, o resto da divisão é o número octal. Por exemplo:
101111
Separando em 3 bits:
101 111
Dividindo cada um por 8:
101 /8
-96 12
5
111 /8
-104 13
7
Justando os restos de divisão, temos o número octal: 57

2. Como converter de base 2 para base 16? \/

R= De 4 em 4 bits, pegue o número binário (base 2) e converta cada grupo de 4 bits para o número hexadecimal correspondente. Por exemplo:
10111111
Separando em 4 bits:
1011 1111
Convertendo cada grupo:
1011 -> B
1111 -> F
O número hexadecimal é: BF

3. Como converter de base 10 para base 2? \/

R= Divida o número decimal (base 10) repetidamente por 2, anotando o resto em cada divisão. Os restos anotados em ordem inversa formam o número binário. Por exemplo:
45
Dividindo por 2:
45 ÷ 2 = 22, resto 1
22 ÷ 2 = 11, resto 0
11 ÷ 2 = 5, resto 1
5 ÷ 2 = 2, resto 1
2 ÷ 2 = 1, resto 0
1 ÷ 2 = 0, resto 1
O número binário é: 101101

4. Como converter de base 10 para base 16? \/

R= Divida o número decimal (base 10) repetidamente por 16, anotando o resto em cada divisão. Converta os restos para a notação hexadecimal. Por exemplo:
255
Dividindo por 16:
255 ÷ 16 = 15, resto 15 (F)
15 ÷ 16 = 0, resto 15 (F)
O número hexadecimal é: FF

5. Como converter de base 16 para base 2? \/

R= Converta cada dígito hexadecimal para seu equivalente binário de 4 bits. Por exemplo:
1F
Convertendo cada dígito:
1 -> 0001
F -> 1111
O número binário é: 00011111

6. Como converter de base 8 para base 2? \/

R= Converta cada dígito octal para seu equivalente binário de 3 bits. Por exemplo:
57
Convertendo cada dígito:
5 -> 101
7 -> 111
O número binário é: 101111

7. Como converter de base 8 para base 10? \/

R= Multiplique cada dígito octal pelo valor de 8 elevado à posição do dígito, somando os resultados. Por exemplo:
57
5 * 8^1 + 7 * 8^0
5 * 8 + 7 * 1
40 + 7 = 47
O número decimal é: 47

8. Como converter de base 16 para base 10? \/

R= Multiplique cada dígito hexadecimal pelo valor de 16 elevado à posição do dígito, somando os resultados. Por exemplo:
1F
1 * 16^1 + F * 16^0
1 * 16 + 15 * 1
16 + 15 = 31
O número decimal é: 31

9. Como converter de base 2 para base 10? \/

R= Multiplique cada dígito binário pelo valor de 2 elevado à posição do dígito, somando os resultados. Por exemplo:
101101
1 * 2^5 + 0 * 2^4 + 1 * 2^3 + 1 * 2^2 + 0 * 2^1 + 1 * 2^0
1 * 32 + 0 * 16 + 1 * 8 + 1 * 4 + 0 * 2 + 1 * 1
32 + 0 + 8 + 4 + 0 + 1 = 45
O número decimal é: 45

10. Como converter de base 10 para base 8? \/

R= Divida o número decimal (base 10) repetidamente por 8, anotando o resto em cada divisão. Os restos anotados em ordem inversa formam o número octal. Por exemplo:
47
Dividindo por 8:
47 ÷ 8 = 5, resto 7
5 ÷ 8 = 0, resto 5
O número octal é: 57
